# Modellierung-von-Kreditausfallrisiken-
Dieses Projekt beschäftigt sich mit der Quantifizierung von Kreditrisiken auf Unternehmensebene und deren Auswirkungen auf ein Portfolio. Ziel ist die Entwicklung eines kreditrisikobasierten Modells zur Schätzung von Ausfallwahrscheinlichkeiten. Hierzu wird zunächst ein Credit-Scoring-Modell verwendet, aus dem implizite Ratings für Unternehmen abgeleitet werden. Aufbauend darauf kommt das Vasicek-Modell zum Einsatz, um systematische Risikofaktoren zu berücksichtigen und Korrelationen zwischen möglichen Ausfällen abzubilden. Durch Stressszenarien werden potenzielle Worst-Case-Ausfallraten analysiert, wodurch ein besseres Verständnis für Extremrisiken entsteht. Das Projekt liefert somit eine strukturierte Grundlage zur Bewertung von Kreditportfolios und unterstützt risikobewusste Investitionsentscheidungen im Unternehmensfinanzierungsumfeld.
